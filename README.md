### Project Title
Bikeshare Data Analysis

### Description
This project is a command-line Python application that analyzes bikeshare usage data for selected U.S. cities.  
It allows users to interactively explore travel patterns by applying filters such as city, month, and day of the week.

The application processes raw bikeshare data and computes descriptive statistics related to travel times, station usage, trip durations, and user demographics, where available.
